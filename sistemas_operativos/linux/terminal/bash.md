## Extras

### Agregar completado

1. Instalar bash-completion:

   ```sh
   pacman -S bash-completion # Arch
   ```

2. Agregar a **_~/.bashrc_**.

   ```rc
   source /usr/share/bash-completion/bash_completion
   ```

3. Recargar terminal.
