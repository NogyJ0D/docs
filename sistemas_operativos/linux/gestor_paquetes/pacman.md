## Comandos

- Actualizar repositorios:
```sh
sudo pacman -Syy
```

- Actualizar paquetes:
```sh
sudo pacman -Syu
```

- Instalar paquete:
```sh
sudo pacman -S <paquete>
```

- Buscar paquete:
```sh
sudo pacman -Ss <paquete>
```

- Eliminar paquete:
```sh
sudo pacman -R <paquete>
```

- Eliminar paquete y dependencias no usadas:
```sh
sudo pacman -Rs <paquete>
```
