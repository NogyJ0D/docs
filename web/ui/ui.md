## Elementos

- Los elementos UI son bloques de construcción visuales que conforman la arquitectura de una interfaz. Contribuyen a la interacción entre usuario y página.
- Garantizan una navegación intuitiva para el usuario. Una buena estructuración ayuda a captar la atención del visitante guiándolo hacia el contenido relevante y acciones específicas.

### Conceptos básicos

- Jerarquía: establecer un orden de importancia ayuda a guiar a los usuarios de manera coherente y efectiva.
- Consistencia: garantiza que los elementos se mantengan uniformes en toda la interfaz.
- Feedback: proporcionar retroalimentación visual después de que un usuario realiza una acción es fundamental para brindar una experiencia satisfactoria.
- Usabilidad: es la facilidad con la que los usuarios pueden interactuar con la interfaz para lograr sus objetivos.
- Espacios en blanco: hay que usarlos adecuadamente para mejorar la legibilidad y el flujo visual de la interfaz.

### Elementos esenciales

- Botones
- Formularios
- Menús de navegación
- Barras laterales
- Íconos
- Barras de desplazamiento
- Tipografía
- Colores
