# DMZ (De-Militarized Zone)

## ¿Qué es?

- Una **zona desmilitarizada** es una subred dentro de una red informática que se encuentra separada de la red interna y que interactúa con el mundo exterior a través de internet.
  - Busca exponer servicios y recursos al exterior sin comprometer la seguridad de la red interna.
- Esta se implementa diseñando tres niveles: red interna, la DMZ y la red externa.
  - Se logra mediante la configuración de un _firewall_ que filtra el tráfico entre las capas.
