## Instalación

### Instalar Go en Linux

1. Descargar última versión.

2. Actualizar y extraer:

    ```sh
     rm -rf /usr/local/go && sudo tar -C /usr/local -xzf gox.xx.x.linux-amd64.tar.gz
    ```

3. Crear variable:

    ```sh
    export PATH=$PATH:/usr/local/go/bin
    ```
