## Comandos

- Eliminar rama local:

  ```sh
  git branch -d [nombre]
  ```

- Deshacer commit:

  ```sh
  git reset HEAD^
  ```

  - Para borrar los cambios agregar --hard
  - Para volver varios agregar el número de commits luego de ^
